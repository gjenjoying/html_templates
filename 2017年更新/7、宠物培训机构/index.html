品牌简介

唯星 最早成立于2005年，后于2009年重新增资成立唯星信息专注于数字营销 的 技术、产品、服务 的 创新 与 整合 ，现已成为中国领先的数字生态服务平台唯星秉承"整合数字资源，技术驱动营销"的理念，

先后从IBM微软华为等知名IT企业引进了许多优秀的研发人才， 另外与复旦、上大等知名院校达成了长期战略合作伙伴关系，成立 "产学研平台" 和 "大数据与新媒体研究中心" 。2013年在上海市科协和虹口区政府帮助下成立了上海市 信息行业首家（计算机）"院士专家工作站"，研发多款唯星自主知识产权的互联网营销技术产品。
